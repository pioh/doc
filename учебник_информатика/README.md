# Инструкция для AI-агента

## Если получил команду "продолжить"

1. Открой `progress.md` - там указана **СЛЕДУЮЩАЯ** глава для написания
2. Напиши эту главу согласно инструкциям ниже
3. Обнови `progress.md` после завершения

---

## Задача проекта

Написать учебник по информатике для первокурсников. Всего **43 главы** в **9 разделах**.

**Текущий прогресс**: смотри в `progress.md`

## Перед началом работы

1. **Прочитай `progress.md`** - узнай какая глава следующая (помечена ← СЛЕДУЮЩАЯ)
2. **Прочитай `spec.md`** - узнай о чём книга и общие требования
3. **Прочитай 1-2 готовые главы** из `chapters/` - поддерживай единый стиль

## Процесс работы

### Шаг 1: Пиши одну главу

- Открой файл главы в `chapters/РР_ТТ_название.md`
- Файл содержит заглушку TODO - замени её полным содержанием
- Объём главы: **3-8 страниц** (5000-15000 символов)

### Шаг 2: Структура главы

```markdown
# Глава X.Y - Название

## Введение

[2-3 абзаца о важности темы и связи с другими главами]

## Основная часть

[Разбей на 3-5 подразделов с заголовками]

### Подраздел 1

[Материал с примерами]

### Подраздел 2

[Материал с примерами]

...

## Ключевые термины

- **Термин 1** - определение
- **Термин 2** - определение

## Контрольные вопросы

1. Вопрос 1?
2. Вопрос 2?
3. Вопрос 3?
```

### Шаг 3: Требования к содержанию

- **Академический стиль** - понятно, но строго
- **Примеры** - минимум 2-3 на главу
- **Термины** - выделяй жирным при первом упоминании
- **Связность** - ссылайся на другие главы где уместно
- **Полнота** - тема должна быть раскрыта целиком

### Шаг 4: После написания главы

Обнови `progress.md`:

1. Поставь ✅ напротив написанной главы
2. Обнови процент прогресса раздела
3. Обнови общий прогресс
4. Добавь короткую заметку (1-2 предложения) о том, что сделано

## Сборка учебника

После написания глав запусти:

```bash
cd /home/tema/doc/учебник_информатика
source venv/bin/activate
python build_script.py
```

Скрипт создаст файл `учебник_информатика.docx`.

## Файловая структура

```
учебник_информатика/
├── README.md              # Эта инструкция
├── spec.md                # Описание проекта и книги
├── progress.md            # Прогресс и заметки
├── build_script.py        # Скрипт сборки DOCX
├── chapters/              # Главы учебника (43 файла)
└── venv/                  # Виртуальное окружение Python
```

## Важно

- **НЕ** пиши несколько глав за раз
- **ВСЕГДА** обновляй `progress.md` после главы
- **ПОДДЕРЖИВАЙ** единый стиль - читай предыдущие главы
- **ПРОВЕРЯЙ** орфографию и пунктуацию
