🎯 КЛЮЧЕВАЯ ИДЕЯ:
   Один AI-агент пишет одну главу в отдельном контексте.
   Координация через служебные файлы (progress, context_summary, agent_notes).

📋 WORKFLOW:
   1. Агент читает: spec.md, progress.md, context_summary.md, agent_notes.md
   2. Агент пишет: одну главу
   3. Агент обновляет: progress, context_summary, agent_notes
   4. Следующий агент повторяет цикл

   ✓ Свежий контекст для каждой главы

   ✓ Полная прослеживаемость прогресса
   ✓ Легко продолжить с любого места
