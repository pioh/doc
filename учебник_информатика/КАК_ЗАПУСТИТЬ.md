# Как запустить агента

## Команда для агента

Просто напиши:

```
продолжить
```

Или более подробно:

```
Прочитай README.md, progress.md и spec.md.
Напиши следующую главу учебника.
```

---

## Что агент сделает автоматически

1. Прочитает `README.md` - увидит инструкцию "продолжить"
2. Прочитает `progress.md` - увидит что следующая глава **01.12 Способы сжатия информации**
3. Прочитает `spec.md` - поймёт требования к главе
4. Прочитает 1-2 готовые главы - поймёт стиль
5. Напишет главу 01.12
6. Обновит `progress.md` (поставит ✅)

---

## После каждой главы

Агент автоматически обновит `progress.md`:

- Поставит ✅ у написанной главы
- Пометит следующую главу как ← **СЛЕДУЮЩАЯ**

Запускай нового агента с той же командой "продолжить".

```bash
source venv/bin/activate

```
